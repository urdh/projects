Major projects

These projects are intended for public consumptions and updated in a fairly regular manner. They should have public issue trackers and proper releases, but that's not a promise.

dotfiles

A Ruby project.

This is my attempt to organize, version-control and share my dotfiles across systems I use. It is inspired by holman's dotfiles, but is written from scratch. The main setup is a ruby program and a small DSL describing a set of “rules” dictating how dotfiles are copies, symlinked or combined.

The repository also contains a bunch of my dotfiles.

On Github.

latexbok

A TeX project.

The “LaTeXbok” is an introductory book about LaTeX written in Swedish. It was originally aimed at students attending the Engineering Physics and Engineering Mathematics programmes at Chalmers, but it is a complete introduction to LaTeX and should be equally useful to other people as well.

Unlike some other LaTeX introductions, the book describes modern best practices and relatively up-to-date packages, which means you won't have to unlearn a bunch of stuff when you discover the TeX community.

On Github. Latest release: n/a.

skdoc

A TeX project.

This is a LaTeX document class designed to generate both package code and package documentation (not entirely unlike the docstrip utility), in only a single run by pdfLaTeX. This is used by the other LaTeX packages and document classes I have been working on, and it is also available on CTAN.

On Github. Latest release: n/a.

skmath

A TeX project.

The skmath LaTeX package provides a multitude of new (and updated) macros designed to ease mathematical typesetting in LaTeX while also enforcing common style conventions. It is also available on CTAN.

On Github. Latest release: n/a.

skrapport

A TeX project.

This document class started out as a toy project, intended to improve my insight into LaTeX document class design while also creating a personal style for reports and similar short documents. Since then, it has grown (perhaps a little too much). It is available on CTAN.

On Github. Latest release: n/a.

Minor projects

These are smaller pet projects mostly maintained for myself or as an experiment. As such, they may go unmaintained for extended periods of time and may not have proper releases. Some of them are useful, some of them are just for fun.

chs-msc-thesis

A TeX project.

This small LaTeX package is an implementation of the Master's thesis style(s) mandated by Chalmers University of Technology.

On Github. Latest release: n/a.

download

A TeX project.

Mostly a proof-of-concept, this LaTeX package was prompted by a question on the TeX stack exchange, and uses common tools such as wget and cURL to fetch online data in LaTeX documents.

On Github. Latest release: n/a.

r-things

An R project.

This is a collection of R scripts simply playing with some data. Might be useful or interesting to someone else, but probably not.

On Github.

streck

A Python project.

Streck is a system implementing a basic credit-based sales terminal. It is written in Python using the Flask framework, but is not intended to be served publically. Instead, it is supposed to run locally and served to a local browser only.

On Github. Latest release: n/a.

vitae

A TeX project.

This is just my resumé. It is available in both Swedish and English, and could probably be used as a base or template for your resumé, assuming you like the way it looks.

On Github.
